# Shelved paper drafts

Drafts that are no longer being pursued for publication, kept for provenance.

## v8_paper_draft.md (πphen perturbational axis)

**Shelved 2026-06-05.** The V8 phenotype axis was pre-registered (OSF pre-reg in
`reports/paper-drafts/v8_osf_preregistration.md`) with a primary Gate 1
(mechanism-class recovery, AMI >= 0.50). When that gate was run on **real** LINCS
L1000 data it FAILED (AMI = 0.13; see `reports/pipeline/v8_real_gate1_v1.md`):
real bulk transcriptional signatures do not recover fine cognition pharmacology.

Per the pre-registration this is an honest negative result, and it is on-thesis:
phenotype, like target-binding affinity and target genetics, underperforms the
mechanism-class clinical track record (the project's central finding). Rather
than reframe a thin negative-result manuscript, the phenotype axis is documented
as a project limitation (see `GAPS_AND_RESEARCH_DIRECTIONS.md`), and the
portfolio focuses on the strong results: the class-prognostic paper and the
V6.A multi-head DTI paper.

The V8 code (`src/mammal_repurposing/cluster_e/`), the real-data gate
(`scripts/92_v8_real_gate1.py`), the OSF pre-registration, and the honest gate
report all remain in the active tree. Only the manuscript draft is shelved.
