# Constructive calibrated class-aware predictor

Turns the critical finding into a usable artifact and answers review-3 items 8 (calibration) + 10 (class as a feature in a constructive model). Reproduced by `scripts/86_constructive_predictor.py`.

## Nested LOCO-CV on the common subset (n = 14)

Leave-one-compound-out logistic regression; the class-prior feature is recomputed inside each fold on the held-out-removed ledger (leakage-clean). Lower Brier is better (0.25 = no-skill at base rate 0.5).

| Model | features | AUROC | Brier |
|---|---|---|---|
| **Class-only (ours)** | class prior | 1.000 | **0.050** |
| Genetics + affinity | target-centric | 0.000 | 0.343 |
| All three | class + genetics + affinity | 1.000 | 0.064 |

**The constructive verdict matches the critical one.** The class-only model is well-calibrated and discriminating (Brier 0.05, AUROC 1.00); adding genetics and affinity changes the Brier score by +0.01 — i.e. the target-centric features add nothing the mechanism-class prior does not already carry, even when a model is free to combine them. A genetics+affinity-only model is poorly calibrated (Brier 0.34). So the recommended predictor is simply the calibrated class prior; the contribution is constructive (a usable, calibrated tool) as well as critical (target-centric inputs are dispensable).
