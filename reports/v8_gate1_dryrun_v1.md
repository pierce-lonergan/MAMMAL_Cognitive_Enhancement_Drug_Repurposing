# V8.4 Gate 1 Dry-Run v1 (synthetic phenotype)

Sanity-checks the V8 Gate 1 mechanism-class recovery pipeline (Leiden + HDBSCAN + AMI/ARI) on synthetic MOFA+-like factors with known ground-truth cluster structure. If the pipeline can't recover 5 well-separated clusters here, it's broken.

## Synthetic dataset

- 5 mechanism classes × 50 compounds = 250 compounds total
- K = 30 latent factors (matches V8.3 MOFA+)
- Noise σ = 0.3
- Class names: cholinergic, catecholaminergic, glutamatergic, trophic_ISR, remyelination
- Class centroids: orthogonal in 30-d space (separable)

## Clustering availability

- Agglomerative (sklearn): ✅ always available
- Leiden (leidenalg): ⏳ not installed
- HDBSCAN (hdbscan): ✅

## Honest caveats

- This is a SYNTHETIC sanity check, not real V8 mechanism-class recovery. Real V8 Gate 1 needs (a) actual MOFA+ joint embedding on real LINCS+JUMP-CP+iPSC-MEA data and (b) ground-truth labels from PRISMA chemistry+literature classification (NOT phenotype-derived; that would be circular).
- Synthetic centroids are orthogonal → cluster recovery is trivially possible. Real signature data has overlap (e.g., stimulants ↔ wake-promoting, AChE-I ↔ α7 nAChR PAM).
- The V8 PRISMA taxonomy is ~30 classes per OSF pre-reg §3, not 5. Real Gate 1 will face a harder K=30 multi-class problem.
- Synthetic noise σ=0.30 is the V8.3 hyperparameter sweep midpoint; real LINCS WTCS τ noise is larger.
